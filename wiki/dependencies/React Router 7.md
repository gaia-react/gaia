# React Router 7

The full-stack web framework GAIA is built on. Provides SSR, file-based routing, loaders, actions, middleware.

## Companion packages (kept in version lockstep)

- `@react-router/node`
- `@react-router/serve`
- `@react-router/dev`
- `@react-router/fs-routes`
- `@react-router/remix-routes-option-adapter`
- `react-router-dom`

The `/update-deps` command upgrades all of these together when react-router is selected.
