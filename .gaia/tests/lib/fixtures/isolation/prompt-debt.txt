question: On main. How should this debt fix be isolated?
header: Branch mode
option1: Create a feature branch in place (Recommended) | Default. Branch is cut from HEAD and the fix works in the current checkout. Simple, predictable, safe.
option2: Create a git worktree | Gives this fix its own separate working copy, cut from main under .claude/worktrees/. You can keep working on your current branch, or run another task, at the same time without the two colliding.
