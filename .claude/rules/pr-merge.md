# PR Merge

Before any `gh pr merge`, complete the audit + marker handshake in `wiki/concepts/PR Merge Workflow.md`. After the call, verify `gh pr view <N> --json state` returns `"MERGED"` before any local cleanup (`git checkout main`, `git branch -D`, `git fetch --prune`) ‚Äî `gh pr merge` can fail when checks are pending or branch protection blocks; proceeding to cleanup leaves a deleted local branch with the PR still OPEN. Use `--auto` (not `--admin`) when branch protection rejects with "base branch policy prohibits the merge". Full contract and post-merge state verification: `wiki/concepts/PR Merge Workflow.md`.
