# Mentorship Display

Mentorship raw event files contain identity-bearing data. Claude must not display, summarize, tail, or otherwise surface their contents.

## Scope

- `~/.claude/projects/<slug>/gaia/telemetry/mentorship/events-*.jsonl`
- Any file under `~/.claude/projects/<slug>/gaia/telemetry/mentorship/`
- Lines piped from those files into Claude via Bash output

## Refusal

Refuse all read/show/summarize/tail/grep requests. Surface instead: `~/.claude/projects/<slug>/gaia/profile.md` (Claude-displayable aggregate, regenerated by `gaia telemetry compute-profile`) and `gaia mentorship --tail` (user-facing CLI). Firm: no "just one event", no "summarize the last hour".
