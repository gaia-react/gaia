# Route & Page Conventions

When editing routes or pages, fetch `wiki/decisions/Thin Routes.md` and `wiki/modules/Pages.md` for the canonical conventions (route thinness, group folders, page dir layout, loader meta pattern, i18n keys, Conform+Zod actions).

For scaffolding a new route, the `new-route` skill owns the templates - it auto-fires when the user asks to create or scaffold a route.
